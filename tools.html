index2txt

A simple console program that will give you a .txt file listing all of your files in a certain folder considering given arguments. This program will be rewritten once I have a better understanding of Java. The rewritten version will also be compatible with MacOS and Linux as well!

convert2youtube

A simple GUI program that will convert any video given from any filetype and codec to an H.264 codec in a .mp4 container (youtube standard filetype). This is basically a GUI for ffmpeg, but makes it much easier to use, and doesn't require any complicated installation or knowledge on ffmpeg.
